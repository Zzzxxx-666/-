# 银行账户管理系统
1.银行职员管理：管理员输入数据建立职员表，添加、删除、修改、查询职员信息，文件存取职员信息，职员查询修改登录密码和个人信息（参考文件操作和通讯录）

2.客户账户管理：银行职员输入数据建立，添加、删除、修改、查询客户资料（客户主账户编号、分类<普通和VIP>、银行卡数量和编号、储蓄贷款余额、电话、开户日期、信用、金融资产、地址等，文件存取信息（参考文件操作、广义表和通讯录）

3.银行卡管理：职员输入数据建立，添加、删除、修改、查询银行卡分账户（编号、类型<借记卡、储蓄卡、信用卡>、借贷余额、利率、开户日期等），文件存取信息（参考文件操作、广义表和通讯录）

4.存取贷业务管理：职员存取贷、转账、利息计算、账户余额、交易时间等,文件存取交易记录（参考航空订票和图书管理）

5.业务查询：职员根据不同关键字（时间段、客户类型、银行卡、金额区间、收入类型、支出类型等）查询存取贷业务明细和汇总信息。（参考航班查询）

6.银行排队管理：银行窗口分VIP窗口和普通窗口，显示各窗口排队和业务办理信息。普通客户和VIP客户，排队取号，VIP优先。一位客户完成业务顺序叫号，客户给职员打分，每日客流统计。（参考银行排队管理）

7.银行网点查询：建立网点地图，查询网点信息，网点路径导航（最短、可行路径），网点增删改（参考校园导游）

8.智能管理：交易异常报警（大额、多笔），银行卡每日/月利息计算，客户统计分析、信用评级和风控审批（收入、支出、借贷比、交易额、金融资产、信用等）、投资顾问、身份核验、生物识别、影像读取等。（2-3项创新）

9. 主界面 ：图形或字符界面，主子菜单，管理员、职员、客户统一登录与管理（主界面设计）
